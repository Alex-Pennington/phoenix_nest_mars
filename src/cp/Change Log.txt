21 Nov 2023

20 Nov 2023
~~~~~~~~~~
 * Gave ALE some much needed TLC
    - Can now set ALE Address to send chat to, or make ALE call (Connect button) from CP Chat
    - Splitter states in ALE dialog are now saved/restored
    - ALE Dialog now saves/restores the last position it was moved to
    - ALE Dialog now saves/restores the last size it was set to
 * Gave Retry Queue some much needed TLC since ALE can use that feature
    - Fixed crash bug in Retry Queue after message(s) deleted
    - Now works properly when enabled and ALE call fails
 * Other dialog windows now save/restore size and position

16 Nov 2023
~~~~~~~~~~
 - Color of received messages is no longer hard coded.  Color of received messages now follows QSS style sheet for QTextBrowser widget.  Basically user/operator can set their own text color in their style sheet.

For example:

QTextBrowser
{
   color: #ffffff; /*white*/
...
...
}
